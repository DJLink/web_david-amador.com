-moz-opacity removed from Firefox 3.5

July 2, 2009 · 1 min · 40 words · David Amador

Well i just found out that Mozzila removed -moz-opacity from Firefox 3.5. It’s been deprecated for a long time and you can how replace it for the standard css property opacity.

So something like -moz-opacity: 0.4 is now opacity: 0.4;
